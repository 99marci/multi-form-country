## Getting Started

## Installation

First be sure to use the correct node version:

`nvm use`

Next, install the required dependencies:

`npm i`

## Running

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
